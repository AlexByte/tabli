Tabli 0.9.8
Mar. 24, 2017
"Mea Culpa" and Minor Bug Fix release:
  • Based on user feedback, the Tabli popout window no longer opens by default when Chrome starts up. If you want this behavior you can still opt-in explicitly via "Preferences..." on the () menu. Thank you to a few loyal users who provided patient, polite, constructive feedback on this.
  • Bug Fix: Fixed a bug where Tabli would sometimes crash at startup if there is no current browser window.
Reminder: If you like Tabli and haven't already done so, please take a few seconds to Rate Tabli on the Chrome Web Store or Tweet about it. Thank You!
